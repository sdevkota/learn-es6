Learn ES6

Let

Let is a new keyword in ES6.It allows you to create variables that are block scoped. The keyword var is still supported but you should avoid using var if you intend to create a local variable.

if (true) { let a = 40; console.log(a); //40 } console.log(a); // undefined

Const

const keyword is used if you would never want to re assign value to a variable declaration. This is similar to contant keyword we have in other server side languages such as C#, Java, and Python

const url= "http://www.google.com"; console.log(url); url= "http://www.yahoo.com" //illegal

Arrow Functions

Arrow function allows you to write function using => arrow syntax instead of using the traditional approach.This is especially handy when passing function as an argument or using a callback functions.

//old way var addMethod=function(test){ return test; } //new way let addMethod=(test)=>test; //traditional approach var total=addMethod(function(a,b){ return a+b; }); //new way let total=addMethod((a,b)=>a+b); //calling a function total(1,2); const abc = name => `Hello ${name}`;

Looping

Es6 allows you to write loop in a modern way just like how we have it in the modern programming langages like C# and JAVA

var arr=[1,2,3,4]; //old way arr.forEach(function(value){ console.log(value); }) //new way for(let value of arr){ console.log(value); }

Template String

Template string builder allows you to concat your string values easily. Please take a look at the example below.

const domain= "google.com"; //new way to concat string literals const fullUrl=`full domain is "https:\\www.${domain}" `; // string interpolation console.log(fullUrl); //expressions inside a string let x=10; console.log(`X is greater than 5? ${x>5?'Yes':'No'}`);

Default function values

With Es6, you can now pass default values of function args if no values are passed.

let test=(name,age=18)=> { return name+":"+age; } test("sudeep");

Object Initilaizer

With Es6, you can initialize the object values before creating an object

let name="sudeep"; let age=27; let obj={ name, age, greet(){ return `Hello ${name}`; } } console.log(obj.greet());

Maps

With Es6, you can create a map using map method.

let map=new Map(); map.set(1, "One"); map.get(1); map.size(); map.has(1); map.clear(); //also Objects/Array can be used as key in Map in JS

Sets

With Es6, you can also create sets using native methods

let test=new Set([1,2,3]);

Classes

With Es6, you can create class natively without creating funtions.

class Test{ Constructor(variable){ this.instanceVariable=variable; } methodName(){ return this.instanceVariable; } }

Constructor

With Es6, you can initialize the object values before creating an object

class Test{ Constructor(variable){ this.instanceVariable=variable; } methodName(){ return this.instanceVariable; } } //invocation let testObject=new Test("hello"); testObject.methodName();

Getters and Setters

With Es6, getters and setters are also available in class

class Test{ Constructor(variable){ this.instanceVariable=variable; } methodName(){ return this.instanceVariable; } get(){ return this.instanceVariable; } set(value){ this.instanceVariable=value; } } //invocation let testObject=new Test(); testObject.set("test"); testObject.get();

Object Initilaizer

With Es6, you can initialize the object values before creating an object and more.

//property shorthand let x=10; let y=20; let obj={x,y}; //ES5 var x = 10; var y = 20; var obj = { x: x, y: y }; //Object Method declaration- fucntion and : is not required let obj={ method1(){ }, method2(){ } } //ES5 let obj={ method1:function(){ }, method2:function(){ } }

Static vs Instance Methods/ Inheritance

class Vehicle{ constructor(make){ this.make=make; } } class Car extends Vehicle{ constructor(make,model){ super(make); this.model=model; } static drive(){ return `driving ${this.make} Model: ${this.model}`; } static honk(){ return "Beep"; } }

Rest Parameters

Rest parameters allow you to quickly pass a list of dynamic array to a function.

let add= (x,y, ...a)=> { let result=0; for(i of [x,y,...a]){ console.log(i); } } add(1,2,3,4)

Spread Operators

Spread operator allow you to quickly display/copy/concat array into series of list items. You can think of this as an opposite of Rest parameters.

let array=[1,2,3] console.log(...array); //another example let array1=[1,2,3] let array2=[4,5,6] let array3= [0, ...array1,...array2];

Destructuring Array/object- Copy and Splits

With Es6, without changing the original array, you can create/assign array and object values to another variables.

//ARRAY let array = [ 1, 2, 3,4 ]; let [ a, , b,c ] = array; console.log(a); //1 console.log(b); //3 console.log(c); //4 //you can also swap values [a,c]=[c,a]; console.log(a); //4 console.log(b); //3 console.log(c); //1 let [ a, , b,c,d=5 ] = array; console.log(d); //5 //OBJECT destruction let test={a:10,b:20,c:30,d:40}; let {a,b,d}=test console.log(a); //10 console.log(b); //20 console.log(d); //40 let {c,e=50}=test; console.log(e); //50
